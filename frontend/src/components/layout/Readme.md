# Componentes de Layout (Layout Components)

Este diretório contém os componentes React de layout do frontend do Imobilay.

## Arquivos e suas Funções

- **[Sidebar.tsx](file:///c:/imobilay/frontend/src/components/layout/Sidebar.tsx)**: Barra lateral de navegação do frontend.
- **[TopBar.tsx](file:///c:/imobilay/frontend/src/components/layout/TopBar.tsx)**: Barra superior de navegação do frontend.

## Funcionamento no Projeto

Os componentes de layout são utilizados para definir a estrutura básica da aplicação Imobilay. Eles garantem que o sistema tenha uma aparência moderna e profissional em todos os navegadores, proporcionando uma experiência de usuário coerente e intuitiva. O uso de componentes funcionais e hooks React garante que o layout seja responsivo e adaptável a diferentes tamanhos de tela.
