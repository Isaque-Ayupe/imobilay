# Componentes de Análise (Analysis Components)

Este diretório contém os componentes React para visualização de análise de imóveis no frontend do Imobilay.

## Arquivos e suas Funções

- **[MetricCell.tsx](file:///c:/imobilay/frontend/src/components/analysis/MetricCell.tsx)**: Componente para visualização de métricas individuais de um imóvel.
- **[PropertyAnalysisCard.tsx](file:///c:/imobilay/frontend/src/components/analysis/PropertyAnalysisCard.tsx)**: Card para exibição completa da análise de um imóvel.
- **[ScoreBar.tsx](file:///c:/imobilay/frontend/src/components/analysis/ScoreBar.tsx)**: Barra de progresso para visualização de pontuação de um imóvel.

## Funcionamento no Projeto

Os componentes de análise são utilizados para exibir as informações geradas pelos agentes especialistas do Imobilay (ex: valuation, investimento, localização). Eles garantem que o usuário receba uma visão clara e objetiva sobre a qualidade e potencial de um imóvel, facilitando a tomada de decisão. O uso de componentes funcionais e hooks React garante que a visualização seja atualizada em tempo real conforme novos dados são processados pelos agentes.
