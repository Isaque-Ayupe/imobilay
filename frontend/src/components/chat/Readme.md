# Componentes de Chat (Chat Components)

Este diretório contém os componentes React para interface de chat com os agentes no frontend do Imobilay.

## Arquivos e suas Funções

- **[ChatInput.tsx](file:///c:/imobilay/frontend/src/components/chat/ChatInput.tsx)**: Campo de entrada de texto para o usuário interagir com o sistema.
- **[MessageBubble.tsx](file:///c:/imobilay/frontend/src/components/chat/MessageBubble.tsx)**: Balão de mensagem para exibição de mensagens enviadas e recebidas no chat.
- **[MessageList.tsx](file:///c:/imobilay/frontend/src/components/chat/MessageList.tsx)**: Lista de mensagens do chat.
- **[PipelineIndicator.tsx](file:///c:/imobilay/frontend/src/components/chat/PipelineIndicator.tsx)**: Indicador visual do progresso do pipeline de execução dos agentes.
- **[TypingIndicator.tsx](file:///c:/imobilay/frontend/src/components/chat/TypingIndicator.tsx)**: Indicador visual de que o sistema está digitando uma resposta.

## Funcionamento no Projeto

Os componentes de chat são a interface de interação principal do Imobilay. Eles permitem que os usuários enviem mensagens, visualizem o histórico de conversas e recebam feedbacks em tempo real sobre o processamento de suas solicitações. O uso de componentes funcionais e hooks React garante que o chat seja interativo e responsivo em todos os navegadores, proporcionando uma experiência de usuário moderna e fluida.
