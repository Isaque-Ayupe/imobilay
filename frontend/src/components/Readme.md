# Componentes React (Components)

Este diretório contém os componentes React reutilizáveis do frontend do Imobilay.

## Diretórios

- **[analysis/](file:///c:/imobilay/frontend/src/components/analysis)**: Componentes para visualização de análise de imóveis.
- **[chat/](file:///c:/imobilay/frontend/src/components/chat)**: Componentes para interface de chat com os agentes.
- **[layout/](file:///c:/imobilay/frontend/src/components/layout)**: Componentes de layout (ex: sidebar, topbar).

## Funcionamento no Projeto

Os componentes React são as peças fundamentais da interface do Imobilay. Eles são organizados em diretórios de acordo com sua função e responsabilidade, facilitando o reuso e manutenção do código. O uso de componentes funcionais e hooks React garante que a lógica de negócio seja separada da lógica de apresentação, permitindo que cada componente seja testado e otimizado individualmente.
