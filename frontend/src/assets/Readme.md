# Ativos Estáticos (Assets)

Este diretório contém os ativos estáticos como imagens e ícones do frontend do Imobilay.

## Arquivos e suas Funções

- **[hero.png](file:///c:/imobilay/frontend/src/assets/hero.png)**: Imagem de destaque (hero image) utilizada na página inicial do Imobilay.
- **[react.svg](file:///c:/imobilay/frontend/src/assets/react.svg)**: Ícone do React.
- **[vite.svg](file:///c:/imobilay/frontend/src/assets/vite.svg)**: Ícone do Vite.

## Funcionamento no Projeto

Os ativos estáticos são utilizados para compor a interface visual do Imobilay. Eles são importados nos componentes React e garantem que o sistema tenha uma aparência moderna e profissional em todos os navegadores. O uso de ativos estáticos permite que o design do sistema seja personalizado e adaptado às necessidades dos usuários.
