# Hooks React Personalizados (Hooks)

Este diretório contém os hooks React personalizados do frontend do Imobilay.

## Arquivos e suas Funções

- **[useChat.ts](file:///c:/imobilay/frontend/src/hooks/useChat.ts)**: Hook para gerenciar o estado e lógica do chat no frontend.
- **[usePipeline.ts](file:///c:/imobilay/frontend/src/hooks/usePipeline.ts)**: Hook para gerenciar o progresso do pipeline de execução dos agentes.

## Funcionamento no Projeto

Os hooks React personalizados são utilizados para extrair a lógica de negócio dos componentes React do Imobilay. Eles garantem que a lógica seja reutilizável e fácil de testar, permitindo que cada hook seja otimizado e mantido individualmente. O uso de hooks funcionais garante que a interface do usuário seja atualizada em tempo real conforme os dados são processados pelos agentes.
