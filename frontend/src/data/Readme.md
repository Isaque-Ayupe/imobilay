# Dados e Mock Files (Data)

Este diretório contém os dados estáticos e mock files utilizados no frontend do Imobilay.

## Arquivos e suas Funções

- **[mock.ts](file:///c:/imobilay/frontend/src/data/mock.ts)**: Dados de exemplo para simular respostas dos agentes no frontend.

## Funcionamento no Projeto

Os dados e mock files são utilizados no desenvolvimento e teste do frontend do Imobilay. Eles permitem simular o comportamento do backend e garantir que a interface seja visualmente correta e interativa antes que os serviços reais estejam integrados. O uso de dados e mock files facilita a colaboração entre desenvolvedores e acelerar o desenvolvimento do sistema.
