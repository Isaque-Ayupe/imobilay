# Definições de Tipos TypeScript (Types)

Este diretório contém as definições de tipos TypeScript utilizadas no frontend do Imobilay.

## Arquivos e suas Funções

- **[index.ts](file:///c:/imobilay/frontend/src/types/index.ts)**: Definições de tipos TypeScript para os modelos de dados utilizados no frontend.

## Funcionamento no Projeto

As definições de tipos TypeScript garantem a qualidade e previsibilidade do código do Imobilay. Eles definem as estruturas de dados utilizadas por todo o sistema, facilitando o desenvolvimento e manutenção do frontend. O uso de tipos estáticos ajuda a evitar erros de lógica e garante que os componentes React se comuniquem de forma clara e estruturada.
