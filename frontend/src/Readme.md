# Código-Fonte do Frontend (Frontend Source)

Este diretório contém o código-fonte principal do frontend do Imobilay.

## Arquivos e suas Funções

- **[App.tsx](file:///c:/imobilay/frontend/src/App.tsx)**: Componente React principal que define a estrutura básica da aplicação.
- **[main.tsx](file:///c:/imobilay/frontend/src/main.tsx)**: Ponto de entrada da aplicação, onde o React é inicializado e montado no DOM.
- **[App.css](file:///c:/imobilay/frontend/src/App.css)**: Estilos CSS globais da aplicação.
- **[index.css](file:///c:/imobilay/frontend/src/index.css)**: Estilos CSS básicos de inicialização.

## Diretórios

- **[assets/](file:///c:/imobilay/frontend/src/assets)**: Ativos estáticos como imagens e ícones.
- **[components/](file:///c:/imobilay/frontend/src/components)**: Componentes React reutilizáveis do projeto.
- **[data/](file:///c:/imobilay/frontend/src/data)**: Dados estáticos e mock files.
- **[hooks/](file:///c:/imobilay/frontend/src/hooks)**: Hooks React personalizados.
- **[types/](file:///c:/imobilay/frontend/src/types)**: Definições de tipos TypeScript.

## Funcionamento no Projeto

O diretório `src/` é onde o desenvolvimento do frontend acontece. Ele organiza os componentes, hooks, tipos e dados de forma modular e clara, facilitando a manutenção e evolução do sistema. O uso de React e TypeScript garante a qualidade e previsibilidade do código, enquanto o Vite proporciona uma experiência de desenvolvimento rápida e eficiente.
