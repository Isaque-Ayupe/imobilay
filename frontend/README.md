# Frontend do Imobilay (Frontend)

Este diretório contém o código-fonte do frontend do Imobilay, desenvolvido com React, TypeScript e Vite.

## Arquivos e suas Funções

- **[package.json](file:///c:/imobilay/frontend/package.json)**: Arquivo de configuração de dependências e scripts do projeto.
- **[vite.config.ts](file:///c:/imobilay/frontend/vite.config.ts)**: Configuração do Vite para o projeto.
- **[tsconfig.json](file:///c:/imobilay/frontend/tsconfig.json)**: Configuração do TypeScript para o projeto.
- **[index.html](file:///c:/imobilay/frontend/index.html)**: Arquivo HTML principal do projeto.

## Diretórios

- **[public/](file:///c:/imobilay/frontend/public)**: Arquivos públicos e ativos estáticos.
- **[src/](file:///c:/imobilay/frontend/src)**: Código-fonte do projeto.

## Funcionamento no Projeto

O frontend do Imobilay é a interface de interação com o usuário. Ele permite que os usuários busquem imóveis, analisem investimentos e recebam feedbacks sobre as suas buscas de forma clara e intuitiva. O uso de React e TypeScript garante a qualidade e manutenibilidade do código, enquanto o Vite proporciona uma experiência de desenvolvimento rápida e eficiente.
