# Arquivos Públicos (Public Assets)

Este diretório contém os arquivos públicos e ativos estáticos do frontend do Imobilay.

## Arquivos e suas Funções

- **[favicon.svg](file:///c:/imobilay/frontend/public/favicon.svg)**: Favicon do projeto Imobilay.
- **[icons.svg](file:///c:/imobilay/frontend/public/icons.svg)**: Sprite de ícones SVG utilizados no frontend.

## Funcionamento no Projeto

Os arquivos públicos são servidos diretamente pelo servidor web e não passam pelo processo de build do Vite. Eles são fundamentais para o visual e identidade visual do Imobilay, garantindo que o sistema tenha uma aparência profissional e coerente em todos os navegadores.
