# Diretivas e SOPs (Directives)

Este diretório contém os SOPs (Standard Operating Procedures) e diretrizes de desenvolvimento do Imobilay.

## Arquivos e suas Funções

- **[01_setup_projeto.md](file:///c:/imobilay/directives/01_setup_projeto.md)**: SOP para configuração inicial do projeto.
- **[02_models.md](file:///c:/imobilay/directives/02_models.md)**: SOP para criação e manutenção de modelos de dados.
- **[03_banco_dados.md](file:///c:/imobilay/directives/03_banco_dados.md)**: SOP para operações e evoluções no banco de dados.
- **[04_camada1_roteamento.md](file:///c:/imobilay/directives/04_camada1_roteamento.md)**: SOP para desenvolvimento da Camada 1 (Input).
- **[05_camada2_orquestrador.md](file:///c:/imobilay/directives/05_camada2_orquestrador.md)**: SOP para desenvolvimento da Camada 2 (Orchestrator).
- **[06_camada3_aprendizado.md](file:///c:/imobilay/directives/06_camada3_aprendizado.md)**: SOP para desenvolvimento da Camada 3 (Learning).
- **[07_frontend.md](file:///c:/imobilay/directives/07_frontend.md)**: SOP para desenvolvimento do frontend.
- **[08_integracao.md](file:///c:/imobilay/directives/08_integracao.md)**: SOP para integração entre as diversas partes do sistema.

## Funcionamento no Projeto

As diretivas são os guias de desenvolvimento do Imobilay. Elas definem os padrões e procedimentos a serem seguidos para garantir a qualidade, consistência e manutenibilidade do código. O uso de SOPs ajuda a reduzir erros e acelerar o desenvolvimento ao fornecer instruções claras e testadas para as tarefas mais comuns.
