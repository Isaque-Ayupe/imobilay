## Funcionamento no Projeto

A suíte de testes do Imobilay é fundamental para garantir a estabilidade e confiabilidade do sistema. Ela permite verificar o funcionamento correto de cada componente individualmente e de forma integrada, facilitando a identificação de bugs e erros de lógica. O uso de testes automatizados ajuda a reduzir o tempo de debug e a manter a qualidade do código a longo prazo.
