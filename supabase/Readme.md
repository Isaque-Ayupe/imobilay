# Configuração do Supabase CLI (Supabase)

Este diretório contém os arquivos de configuração do Supabase CLI para o projeto Imobilay.

## Arquivos e suas Funções

- **[config.toml](file:///c:/imobilay/supabase/config.toml)**: Arquivo de configuração do Supabase CLI, definindo parâmetros do banco de dados local.
- **[.gitignore](file:///c:/imobilay/supabase/.gitignore)**: Arquivo que define os arquivos e diretórios que devem ser ignorados pelo Git.

## Diretórios

- **[migrations/](file:///c:/imobilay/supabase/migrations)**: Scripts SQL para migrações do Supabase CLI.

## Funcionamento no Projeto

O diretório `supabase/` é utilizado pelo Supabase CLI para gerenciar as migrações e configurações do banco de dados local. Ele garante que todos os desenvolvedores tenham um ambiente de banco de dados idêntico e facilita o deploy de mudanças de esquema para a produção.
