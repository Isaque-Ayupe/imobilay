# Migrações do Supabase CLI (Supabase Migrations)

Este diretório contém os scripts SQL de migração para o Supabase CLI no projeto Imobilay.

## Arquivos e suas Funções

- **[20260324120000_001_initial.sql](file:///c:/imobilay/supabase/migrations/20260324120000_001_initial.sql)**: Script SQL inicial que cria a estrutura básica do banco de dados no Supabase.

## Funcionamento no Projeto

As migrações do Supabase CLI são utilizadas para versionar as mudanças no esquema do banco de dados do Imobilay. Elas permitem que todos os desenvolvedores tenham a mesma estrutura de tabelas, índices e restrições no seu ambiente local e facilitam o deploy para a produção de forma controlada e auditável.
