# Configuração de Agentes (Agents)

Este diretório contém a configuração e habilidades (skills) para os agentes de IA do Imobilay.

## Diretórios

- **[skills/](file:///c:/imobilay/.agents/skills)**: Habilidades (skills) disponíveis para os agentes.

## Funcionamento no Projeto

O diretório `.agents/` é utilizado para configurar o comportamento dos agentes de IA no Imobilay. Ele define quais habilidades os agentes possuem e como eles devem interagir com o sistema e os usuários. O uso de habilidades especializadas permite que os agentes realizem tarefas complexas (ex: análise de banco de dados, sugestões de investimento) de forma eficiente e confiável.
