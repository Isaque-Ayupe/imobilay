# Habilidade: Melhores Práticas de Supabase/PostgreSQL (Supabase Postgres Best Practices)

Este diretório contém a habilidade (skill) para análise e sugestões de melhores práticas de banco de dados Supabase/PostgreSQL no Imobilay.

- **[CLAUDE.md](file:///c:/imobilay/.agents/skills/supabase-postgres-best-practices/CLAUDE.md)**: Instruções para o agente Claude utilizar esta habilidade.
- **[README.md](file:///c:/imobilay/.agents/skills/supabase-postgres-best-practices/README.md)**: Documentação principal da habilidade.
- **[SKILL.md](file:///c:/imobilay/.agents/skills/supabase-postgres-best-practices/SKILL.md)**: Definição técnica da habilidade.

## Diretórios

- **[references/](file:///c:/imobilay/.agents/skills/supabase-postgres-best-practices/references)**: Documentação de referência detalhada sobre as melhores práticas.

## Funcionamento no Projeto

A habilidade `supabase-postgres-best-practices` é utilizada pelos agentes de IA do Imobilay para garantir que o banco de dados seja projetado e mantido seguindo as melhores práticas do Supabase e PostgreSQL. Ela fornece diretrizes claras sobre performance, segurança, integridade de dados e escalabilidade, garantindo que o sistema seja robusto e eficiente a longo prazo.
