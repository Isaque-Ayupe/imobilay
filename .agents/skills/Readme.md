# Habilidades de Agentes (Agent Skills)

Este diretório contém as habilidades (skills) especializadas para os agentes de IA do Imobilay.

## Diretórios

- **[supabase-postgres-best-practices/](file:///c:/imobilay/.agents/skills/supabase-postgres-best-practices)**: Habilidade para análise e sugestões de melhores práticas de banco de dados Supabase/PostgreSQL.

## Funcionamento no Projeto

As habilidades de agentes são utilizadas para fornecer conhecimento especializado aos agentes de IA do Imobilay. Cada habilidade é focada em uma área específica (ex: melhores práticas de banco de dados) e garante que o sistema tenha acesso às melhores práticas e recomendações para o seu desenvolvimento e manutenção. O uso de habilidades especializadas permite que os agentes realizem tarefas complexas de forma eficiente e confiável.
